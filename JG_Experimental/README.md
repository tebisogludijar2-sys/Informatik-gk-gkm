# JG_Experimental – einfacher RoboCup Rescue Line 2026 Parkour-Generator

Dieses Web-Tool erzeugt einen **einfachen, möglichst machbaren** Beispiel-Parkour auf Grundlage zentraler Feldregeln aus den RoboCupJunior Rescue Line 2026 Regeln.

## Start

Einfach `index.html` im Browser öffnen.

## Funktionen

- Generiert eine Tile-Sequenz mit mindestens 8 Wettbewerbs-Tiles (ohne Start/Goal)
- Nutzt eine vorsichtige Hazard-Verteilung (niedrige bis moderate Schwierigkeit)
- Zeigt eine visuelle Vorschau inkl. Evacuation Zone
- Zeigt eine textuelle Zusammenfassung und die komplette Tile-Sequenz
- Button „Parkour neu generieren“ für neue Varianten

## Hinweis

Das Tool bildet einen praxisnahen, **vereinfachten** Generator ab, damit der Parkour gut machbar bleibt.
